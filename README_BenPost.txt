README_BenPost: BenPost.c is a rudimentary post processor for CU-BEN.

Usage:
Keep in same folder as CU-BEN's results*.txt output files
After creating and running BenPost executable, a "master.pvd" file will output for ParaView accessibility
A folder will also be output named "vtu_files" which will contain a "slave_*.vtu" file for each time step of the CUBEN simulation
This folder can be opened in ParaView to be saved as a .vtk file
